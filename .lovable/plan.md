
## Clean Minimalist Developer Portfolio — Dark Mode

### Design
- **Theme**: Dark background (#0a0a0a), white/light gray text, subtle accent color (muted blue or cyan for links/highlights)
- **Typography**: Clean sans-serif, generous spacing, large hero text
- **Layout**: Single-page scrolling, full-width sections with max-width content container

### Sections

1. **Hero** — Full-height intro with your name, title ("Software Developer"), a one-liner tagline, and CTA buttons (View Work / Contact Me)

2. **About** — Brief bio paragraph with a subtle card or side-by-side layout, keeping it concise

3. **Skills** — Grid of tech skills displayed as minimal pills/badges grouped by category (Frontend, Backend, Tools)

4. **Experience** — Timeline-style list of roles with company, title, date range, and a short description each

5. **Projects** — Card grid (2-3 featured projects) with title, description, tech tags, and links to GitHub/live demo

6. **Contact** — Simple contact form (name, email, message) with social links (GitHub, LinkedIn, email)

### Navigation
- Fixed top navbar with smooth-scroll links to each section
- Subtle blur/glass effect on scroll

### Interactions
- Smooth scroll between sections
- Subtle fade-in animations on scroll using CSS
- Hover effects on project cards and buttons
